    Submit an Event

    There are two ways to submit your event. If you want to upload images, you must be signed into a Google account. If you do not want to sign in, you can submit image links instead.

    Featured card text (Google Form): add three short-answer questions so each event can supply the three lines shown on site cards. Use response validation (character count — maximum) so entries stay within limits:

    • Card line 1 — name or headliner (e.g. Justin Terry). Max 36 characters.
    • Card line 2 — what the event is (e.g. Valentines Dance). Max 48 characters.
    • Card line 3 — date line as residents should read it, or leave blank and the site will show the event date as weekday day month 2-digit year (e.g. Wed 10 Feb 27). Max 32 characters.

    Map these fields to cardLine1, cardLine2, and cardLine3 on each event in mmhp-master-data.json (or use Export CSV on the calendar page after merging Sheet data).

    The Google Form buttons are not linked yet. Edit contents/submit.html and set each form button’s href to your published URL (for example https://forms.gle/…).
