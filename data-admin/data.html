This page is the only web-based editor for master data (password plus the optional Admin link on the public home page).

Desktop only

It is not available on phones or tablets; those devices are redirected to the home page. Use a desktop or laptop browser. Bookmark data-admin/data.html (browse.html redirects here).

Source file and on-screen limits

Source file: assets/data/json/mmhp-master-data.json. Tables show up to 500 rows on screen; exports include all rows. After edits, use per-collection Export … to CSV when you need a spreadsheet slice.

Saving your work

Use Save changes to the Master Data File… to write the full JSON back through your browser’s save dialog. If you close or leave the tab without saving, your browser should warn you that unsaved changes may be lost.

Google Calendar

The public calendar iframe is maintained in Google Calendar directly (one recurring event per activity, not from this JSON). Recurring activities below drive the site’s left sidebar only.

Features and forms

On the public site, card line 2 comes from the activity name tied to your Activity choice; it is not stored as a separate field on the feature row. Listing title (eventName) is filled automatically from the short description plus activity when you Apply. Treat every field as required unless the form explicitly allows empty values (for example No end time on features).

Tables (sort and filter)

All tables live in the bordered box between the message center and this help section: use the box’s right-hand scrollbar to move through long data while the page header and message center stay above that area. Use the leading checkbox on each row (or the header checkbox to select all visible rows, up to 500) and Delete selected to remove several rows at once. Click a column title to sort: ascending, then descending, then back to the default order. Click the ▼ button on a column to open a filter checklist (similar to Excel). Export CSV still exports the full collection, not the filtered on-screen view.
