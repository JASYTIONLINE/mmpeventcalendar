Master data

Edit, delete, export by collection (password required)

This page is the only web-based editor for master data (password + optional corner Admin link on the public home page). It is not available on phones or tablets — those devices are sent to the home page; use a desktop or laptop browser. Bookmark data-admin/data.html — browse.html redirects here. Source: assets/data/json/mmhp-master-data.json. Up to 500 rows per table on screen; export includes all rows. Use Save changes to master JSON after edits, or Export … to CSV per collection. Events: Card line 2 on the public site is the activity name from your Activity choice (it is not stored on the event). Listing title (eventName) is filled automatically from event title + activity when you Apply. Note: Treat every field in the form as required before Apply. Only use options that explicitly allow empty values (for example, No end time on events). If you edited or deleted rows and try to close the tab or leave this page without using Save changes to the Master Data File, your browser will warn you that unsaved changes may be lost.
